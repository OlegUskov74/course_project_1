# Приложение для анализа транзакций, которое находятся в Excel-файле.

### Описание

Приложение будет генерировать JSON-данные для веб-страниц, формировать Excel-отчеты, а также предоставлять другие сервисы.








## Тестирование

### Покрытие кода тестами

Для проверки покрытия кода тестами можно использовать `pytest-cov`:

```sh
pytest --cov=src
```

Где `src` — папка с исходным кодом.

### Логирование и отчеты

Для генерации HTML-отчёта о покрытии выполните:

```sh
pytest --cov=src --cov-report=html
```
Coverage report: 90%
